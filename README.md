# Normtech Site

Minimal landing page for Normtech with terms, privacy policy, and support pages.

## Pages

- `index.html` — Landing page
- `terms.html` — Terms & Conditions
- `privacy.html` — Privacy Policy
- `support.html` — Support contact

## Deployment

Static HTML — deploy anywhere (GitHub Pages, Vercel, Netlify, Cloudflare Pages).

For GitHub Pages: Settings → Pages → Source: Deploy from branch → `main` / `/ (root)`.

## Canva App Listing URLs

Once deployed to e.g. `normtech.io`:

- **Company URL:** `https://normtech.io`
- **Terms & Conditions:** `https://normtech.io/terms.html`
- **Privacy Policy:** `https://normtech.io/privacy.html`
- **Support:** `https://normtech.io/support.html`
